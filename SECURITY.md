# Security Policy

dotden is pre-v1; there are no supported released versions yet. Security fixes land on the default branch.

## Reporting a vulnerability

Please **do not** open a public issue for security problems.

Report privately via [GitHub Security Advisories](https://github.com/mathewcst/dotden/security/advisories/new).

Include steps to reproduce and the affected component (`apps/desktop`, `apps/web`, …) where possible.

We'll acknowledge your report and keep you updated on the fix.
